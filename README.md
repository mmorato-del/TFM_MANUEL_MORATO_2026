Código Python / VHDL del Trabajo de Fin de Máster de Manuel Morato Miguel, a 8 de junio de 2026

EEGNet_Eneriz_2.py : Funciones básicas para definir el modelo EEGnet con las adaptaciones propuestas en la Tesis Doctoral de Daniel Enériz Aorta, 
el entrenamiento con validación cruzada y de sujeto específico.

CV_implementation.ipynb, FFT_implementation.ipynb : Implementación del entrenamiento de los modelos en representación temporal y frecuencial con 
la base de datos Physionet Motor Movement/Imagery Dataset

timeWindow.ipynb : Evaluación de los modelos ante el desplazamiento de las ventanas temporales.

Códigos .vhd : módulos VHDL del co-procesador reconfigurable para interfaces cerebro-computador (Brain Computer Interface BCI) portátiles, capaz de 
realizar la transformada rápida de Fourier de un número programable de señales electroencefalográficas (EEG)
